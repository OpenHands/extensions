# Slack Responder

Set up an automated Slack responder that monitors channels for engineering questions and requests, then responds with helpful information or takes action on behalf of the team.

## Triggers

- `/slack-responder`
- `/setup-slack-bot`

## What It Does

1. Monitors Slack channels for trigger phrases
2. Responds to engineering questions
3. Takes action on requests
4. Integrates with OpenHands automations
