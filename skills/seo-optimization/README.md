# SEO Optimization

Analyze and improve website SEO including meta tags, structured data, performance, and content optimization for better search engine rankings.

## Triggers

- `/seo-audit`
- `/optimize-seo`

## What It Does

1. Audits meta tags and content structure
2. Checks technical SEO requirements
3. Validates and implements structured data
4. Analyzes Core Web Vitals
5. Provides specific code fixes
