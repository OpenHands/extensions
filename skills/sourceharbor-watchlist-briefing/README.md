# SourceHarbor Watchlist Briefing

This skill packages the watchlist-briefing workflow from [SourceHarbor](https://github.com/xiaojiou176-open/sourceharbor) for OpenHands.

It is meant to feel like a **plugin-grade operator recipe**:

- start from one watchlist
- load the current briefing/story payload
- answer one operator question with evidence
- finish with one concrete next action

Use it when you want a source-first briefing workflow that can reuse SourceHarbor MCP or HTTP surfaces without pretending the product is already a hosted SaaS.
