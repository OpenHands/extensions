# arXiv Literature Search

Academic literature search guidance for agents working with arXiv, OpenAlex, and Semantic Scholar.

Use this skill when a task needs paper discovery, related-work collection, arXiv-safe rate limiting, OpenAlex/Semantic Scholar metadata enrichment, or browser fallback for official paper pages.

The skill emphasizes:

- API-first literature discovery
- arXiv-friendly throttling and retry behavior
- OpenAlex API key usage
- provider-level caching and deduplication
- browser control only when APIs are insufficient

See [SKILL.md](SKILL.md) for the full workflow.
