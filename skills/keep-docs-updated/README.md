# Keep Docs Updated

Keep documentation in sync with code changes. Automatically updates READMEs, API docs, inline comments, and other documentation when code changes are made.

## Triggers

This skill is activated by the following commands:

- `/update-docs`
- `/sync-docs`

## What It Does

1. Identifies documentation that's out of sync with code
2. Updates READMEs, API docs, and inline documentation
3. Generates new documentation for undocumented code
4. Keeps examples and tutorials current

## Documentation Types Handled

- README files and quick start guides
- API documentation and function signatures
- Inline comments and docstrings
- Usage examples and tutorials
- Configuration documentation
