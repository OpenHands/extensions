# Plain English content skill

This skill helps OpenHands write and edit clear, accessible prose in a plain English content style.
It focuses on active voice, front-loaded structure, sentence case, accessible formatting, and public-facing guidance conventions.

Use it for:

- rewriting reports, research summaries and guidance in plain English
- checking public-facing documentation for clarity and accessibility issues
- briefing another agent to produce clear, accessible prose

The skill was imported and adapted from this public gist: <https://gist.github.com/fofr/505e225f9bf5e839d30c12ba6bfa0be2>.
