# Internationalization

Implement internationalization (i18n) and localization (l10n) support including string extraction, translation management, locale handling, and RTL support.

## Triggers

- `/i18n`
- `/internationalize`
- `/add-translations`

## What It Does

1. Audits hardcoded strings
2. Sets up i18n framework for your stack
3. Extracts strings to translation files
4. Implements locale-aware formatting
5. Adds RTL language support
