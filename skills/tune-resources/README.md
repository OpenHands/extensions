# Tune Resources

Optimize application memory and CPU usage by analyzing resource consumption patterns, identifying inefficiencies, and implementing performance improvements.

## Triggers

- `/tune-resources`
- `/optimize-memory`
- `/optimize-cpu`

## What It Does

1. Analyzes code for resource-intensive operations
2. Identifies memory leaks and excessive allocations
3. Finds CPU hotspots and inefficient algorithms
4. Implements targeted optimizations
