# Code Review

Structured code review covering style, readability, and security concerns with actionable feedback. Posts inline review comments via the GitHub API with priority labels. Use when reviewing pull requests or merge requests.

This skill combines review guidelines with instructions for posting structured inline comments via the GitHub API (previously the separate `github-pr-review` skill).

## Triggers

This skill is activated by the following keywords:

- `/codereview`
- `/github-pr-review`

See [SKILL.md](./SKILL.md) for the full skill content including review scenarios, GitHub API posting instructions, priority labels, and suggestion syntax.
