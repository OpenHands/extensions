# Fix Merge Conflicts

Resolve git merge conflicts intelligently by understanding the intent of both changes. Handles code conflicts, configuration file merges, and lock file reconciliation.

## Triggers

- `/fix-conflicts`
- `/resolve-conflicts`

## What It Does

1. Analyzes both sides of each conflict
2. Understands the intent behind each change
3. Resolves conflicts while preserving both intents
4. Handles lock file regeneration when needed
