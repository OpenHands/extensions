# Zero-Day Response

Rapidly respond to zero-day vulnerabilities by assessing exposure, implementing mitigations, and applying patches. Designed for urgent security incidents.

## Triggers

- `/zero-day`
- `/cve-response`
- `/urgent-vuln`

## What It Does

1. Assesses if your codebase is affected
2. Evaluates severity and attack vectors
3. Implements immediate mitigations
4. Applies patches or workarounds
5. Verifies the fix is effective
