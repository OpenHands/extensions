# /zero-day

Respond to a zero-day vulnerability.

## Usage

```
/zero-day <CVE-ID>
/zero-day <vulnerability-description>
```

## Examples

```
/zero-day CVE-2024-1234
```
Assesses exposure to a specific CVE and provides response plan.

```
/zero-day "Log4j remote code execution"
```
Searches for and responds to a described vulnerability.
