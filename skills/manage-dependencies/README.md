# Manage Dependencies

Manage and upgrade project dependencies including security updates, breaking change handling, and version compatibility. Supports npm, pip, cargo, and other package managers.

## Triggers

- `/update-deps`
- `/manage-dependencies`
- `/upgrade-packages`

## What It Does

1. Audits outdated packages and security vulnerabilities
2. Plans update strategy based on risk
3. Handles breaking changes in major updates
4. Updates lock files for consistency
5. Validates changes with tests
